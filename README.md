# What's This/这是啥

This extension makes it possible for you to learn OI efficiently using VS Code.

此扩展可以让你在VS Code中高效的学习OI，是您学习中的好帮手！

# Current Features/目前功能

根据您的选择，自动导入您所需的模板！只需轻松改变部分代码，便可AK全场！
